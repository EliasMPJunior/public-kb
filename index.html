InfoBIM Framework
Link
A modular runtime environment for Engineering Automation that bridges the gap between static BIM data and dynamic AI agents. InfoBIM provides a machine-readable catalog of tools via its Discovery Protocol, turning ad-hoc OpenBIM scripts into a repeatable production line of capabilities. It enables LLMs to orchestrate automated logic checking, IDS-native rules, and descriptive report generation directly from IFC models, acting as the capability engine for BIM automation.
